# Decision Log

## 2026-01-16 — 日本語対応エージェント設定を追加

- 決定: GitHub Copilot Coding Agent の動作をカスタマイズする日本語対応エージェント設定を追加。
- 理由: Copilot Agent を使用した開発プロセスで、日本語でのコミュニケーション、事前プラン策定、ユーザーへの質問を標準化し、高品質な開発プロセスを確立するため。
- 実装: `.github/agents/japanese-agent.md`
- 主な機能:
  - PR タイトル、説明欄、コミットメッセージを日本語で記述
  - 実装前に必ずプランをチェックリスト形式で作成
  - 不明点がある場合はユーザーに質問
  - コード品質、セキュリティ、レビュープロセスのベストプラクティスを含む

## 2025-12-20 — PR Size Labeler を追加

- 決定: PR の行数に基づく `size/XS`〜`size/XL` ラベル付与を実装。
- 理由: 迅速に PR の相対的な作業量を判断できるため。実装は `.github/workflows/pr-size-label.yml`。
- 閾値: XS <10, S <50, M <200, L <500, XL >=500
- 実装メモ: `git diff --numstat base...head` で追加・削除を合算して算出。既存のサイズラベルは削除して最新を付与する。
